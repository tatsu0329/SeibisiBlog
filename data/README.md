# データディレクトリ

このディレクトリには、過去問データとブログ記事データを配置します。

## 分類について

過去問の分類は、[一般社団法人日本自動車整備振興会連合会の公式サイト](https://www.jaspa.or.jp/mechanic/past/)の分類に準拠しています。

- 一級小型自動車（筆記）
- 一級小型自動車（口述）
- 二級ガソリン自動車
- 二級ジーゼル自動車
- 二級自動車シャシ
- 三級自動車シャシ

## 過去問データ形式（JSON）

```json
{
  "year": "2024",
  "categorySlug": "2級ガソリン自動車",
  "level": 2,
  "category": "ガソリン",
  "examType": "学科",
  "questions": [
    {
      "id": "1",
      "questionNumber": 1,
      "question": "問題文...",
      "choices": ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
      "correctAnswer": 0,
      "explanation": "解説文...",
      "tags": ["タグ1", "タグ2"]
    }
  ]
}
```

## ブログ記事形式（Markdown）

```markdown
---
title: 記事タイトル
excerpt: 記事の抜粋
date: 2024-01-15
tags: [タグ1, タグ2]
category: 解説
---

# 記事本文

Markdown形式で記述
```

## 実装方法

現在、`lib/data.ts` でサンプルデータを返しています。実際のデータを使用するには：

1. このディレクトリにデータファイルを配置
2. `lib/data.ts` を更新してファイルシステムから読み込む
3. または、CMSやデータベースに移行
